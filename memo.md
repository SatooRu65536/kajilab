# 単語帳
### ノルム
携帯の持ち方による違いなどをなくす  
> ベクトル量で表される加速度をx軸、y軸、z軸の成分に分解し、それぞれの成分の二乗を足し合わせたものの平方根


```
√(ax^2 + ay^2 + az^2)
```
